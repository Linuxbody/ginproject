##### 项目基本目录
```
╰─$ tree
.
├── conf
│   └── config.yml
├── controller
│   └── user.go
├── logs
│   └── ginWeb.log
├── main.go
├── middleware
│   └── logger.go
├── models
├── routers
│   └── router.go
└── utils
    └── settings.go
    
    
conf：用于存储配置文件
middleware：应用中间件
models：应用数据库模型
utils：第三方包
routers: 路由逻辑处理
controller: 控制器
logs: 日志目录
 
main.go   :  主方法 
```
